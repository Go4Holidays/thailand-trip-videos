# Thailand Trip Videos

A Python project for creating and editing short travel videos from Thailand trip clips. Automatic video compilation with transitions, effects, and music integration for YouTube Shorts, TikTok, and Instagram Reels.

## Features

- Automatic video compilation
- - Add transitions and effects
  - - Background music integration
    - - Video trimming and cropping
      - - Text overlays for locations
        - - Speed control and effects
         
          - ## Project Structure
         
          - README.md, requirements.txt, config/settings.json, src/ (video_editor.py, clip_processor.py, effects.py), samples/sample_clips, assets/bgm.mp3, output/
         
          - ## Installation
         
          - pip install -r requirements.txt
         
          - ## Quick Start
         
          - python example_usage.py
